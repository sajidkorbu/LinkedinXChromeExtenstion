LinkedIn Post Analyzer Settings

How to get your OpenAI API Key:

  1. Go to OpenAI API Keys
  2. Click "Create new secret key"
  3. Give it a name (e.g., "LinkedIn Post Analyzer")
  4. Copy the key (starts with 'sk-')
  5. Paste it below and click Save

Security Note: Your API key is stored securely on your device only. While the key is used to make direct API calls to OpenAI when analyzing posts, it is never shared with us or any third parties. Each user of this extension needs to provide their own API key for the analysis to work.

The extension only uses your key to send post content to OpenAI for analysis, and this happens directly from your browser to OpenAI's servers.
